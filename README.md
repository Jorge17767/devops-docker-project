# DevOps Docker Project

This is my first DevOps project using Docker.

## Project description

Simple web application deployed with Docker using Nginx.

## Technologies used

- Docker
- HTML
- Nginx

## How to run the project

Build the Docker image:

docker build -t devops-project .

Run the container:

docker run -p 8080:80 devops-project

Open the browser:

http://localhost:8080
